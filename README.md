# The Fat Pitch Portfolio Dashboard 🚀

Live portfolio dashboard tracking my investments vs S&P 500.
Built with Python + Plotly. Updated weekly.

## Live Dashboard
https://thefatpitch.github.io/thefatpitch-portfolio/

## What It Shows
- Performance vs S&P 500 since inception
- Monthly returns by instrument (Stocks, ETFs, Options, Cash)
- Annual returns breakdown
- Current stock holdings
- Options portfolio

## How It Works
1. Download weekly report from IBKR
2. Run convert_ibkr_to_csv.py locally
3. Run generatedashboard.py to generate index.html
4. Push index.html to GitHub
5. Dashboard updates live automatically

## Tech Stack
- Python + Pandas
- Plotly.js
- GitHub Pages
